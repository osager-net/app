Как взаимодействовать с блогом через электронную почту

взаимодействия с блогом используйте адрес: osager-net@mail.ru. Ниже — подробные инструкции для разных действий.

1. Как добавить свой пост ( Он будет помещен в соответствующий тег, а если его нету: добавим в "Другое" )

Чтобы опубликовать пост, отправьте письмо на указанный адрес. В письме обязательно укажите следующую информацию — в том порядке, который приведён ниже:
ID поста — уникальный идентификатор поста. При его составлении: используйте только буквы латинского алфавита ( верхнего и нижнего регистра: A–Z, a–z ). Разрешается использовать символы, которые есть на обычной клавиатуре ( например: -, _, цифры 0–9 и т. д. ). Не используйте «невидимые» символы или спецсимволы, скопированные из интернета, или что то похожее.
Заголовок поста — напишите его на новой строке после ID.
Текст поста — разместите его после заголовка, отделив пустой строкой. Пишите развёрнуто, форматирование ( списки, абзацы ) сохраняется.
Информация об авторе — в конце письма укажите, кто вы ( ваше имя, псевдоним или никнейм ).

2. Как оставить комментарий к посту

Чтобы прокомментировать уже существующий пост, отправьте письмо на тот же адрес. Структура письма:
ID поста, к которому вы хотите оставить комментарий ( укажите его в самом начале письма ).
Текст комментария — напишите его после ID, отделив пустой строкой. Будьте вежливы и по делу.
Информация о комментаторе — в конце укажите, кто вы.

3. Как добавить тег

Чтобы предложить новый тег для поста, отправьте письмо по тому же адресу. В письме:
Начните с фразы, указывающей на ваше намерение добавить тег ( например: «Хочу добавить тег» или «Предлагаю тег» ).
На следующей строке напишите название тега. В названии тега разрешены пробелы, и можно использовать буквы русского и латинского алфавита, цифры и обычные символы с кклавиатуры
