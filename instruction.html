Как взаимодействовать с блогом

для взаимодействия с блогом используйте адрес: https://osager-net.github.io/app/forma.html. Ниже — подробные инструкции для разных действий.

1. Как добавить свой пост ( Он будет помещен в соответствующий тег, а если его нету: добавим в "Другое" )

Чтобы опубликовать пост, отправьте форму для создания поста из адреса. В письме обязательно укажите следующую информацию — в том порядке, который приведён ниже:
ID поста — уникальный идентификатор поста. При его составлении: используйте только буквы латинского алфавита ( верхнего и нижнего регистра: A–Z, a–z ). Разрешается использовать символы, которые есть на обычной клавиатуре ( например: -, _, цифры 0–9 и т. д. ). Не используйте «невидимые» символы или спецсимволы, скопированные из интернета, или что то похожее.
Заголовок поста — напишите его на новой строке после ID.
Текст поста — разместите его после заголовка, отделив пустой строкой. Пишите развёрнуто, форматирование ( списки, абзацы ) сохраняется.
Осайдберы — дополнительные ссылки на видео, фото и тд. Можно добавить от 0 до 10 осайдберов после текста.
Оссайтеры — видео и фото. Прикрепите в письмо оссайтеры с такими вариантами кол — во: 0 видео / фото оссайтеров, 1 видео оссайтер, 1 фото оссайтер, 2 видео & фото оссайтеров. Видео оссайтер не должен быть больше 20 МБ, а фото оссайтер: не больше 1080p качества. Информация об авторе — в конце письма укажите, кто вы ( ваше имя, псевдоним или никнейм ).

2. Как оставить комментарий к посту

Чтобы прокомментировать уже существующий пост, отправьте заявку на тот же адрес. Структура письма:
ID поста, к которому вы хотите оставить комментарий ( укажите его в самом начале письма ).
Текст комментария — напишите его после ID, отделив пустой строкой. Будьте вежливы и по делу.
Информация о комментаторе — в конце укажите, кто вы.

3. Как добавить тег

Чтобы предложить новый тег для поста, отправьте форму по тому же адресу. В письме:
Напишите название тега. В названии тега разрешены пробелы, и можно использовать буквы русского и латинского алфавита, цифры и обычные символы с кклавиатуры
